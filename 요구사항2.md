# 프로젝트 개요

## [목적]
동네 식자재 마트의 온라인 주문 시스템을 구축하여, 고객이 모바일/웹으로 편리하게 주문하고 배달 또는 픽업으로 받을 수 있는 서비스를 제공한다.

**핵심 가치:**
- 동네 주민들의 편리한 장보기
- 전화 주문 부담 감소
- 배달/픽업 주문 관리 효율화

## [타겟 사용자]
* **주부층**: 집에서 편하게 장보기
* **직장인**: 퇴근 후 픽업
* **시니어층**: 큰 글씨, 간단한 UI

---

# 기술 스택

## [프론트엔드]
* **Web**: HTML, CSS, JavaScript (또는 React)
* **모바일**: 반응형 웹 (PWA 가능)
* **스타일**: Tailwind CSS 또는 Bootstrap

## [백엔드]
* **언어**: Node.js + Express 또는 Python + Flask
* **데이터베이스**: 
  - SQLite (소규모) 또는 PostgreSQL
  - Google Sheets (초기 테스트용 가능)

## [결제]
* **PG사**: 토스페이먼츠 (간단한 연동)
* **간편결제**: 카카오페이, 토스

## [배포]
* **호스팅**: Vercel, Netlify (프론트)
* **백엔드**: Railway, Render (무료 티어)

# 핵심 기능 요구사항 (MVP)

## [1. 회원 관리 - 최소화]

### 회원가입/로그인
* **필수 정보만**: 이름, 전화번호, 비밀번호
* **간편 로그인**: 카카오 로그인 (선택사항)
* **비회원 주문 가능**: 전화번호만 입력

### 마이페이지
* 배송지 1개 저장 (기본 주소)
* 주문 내역 조회
* 전화번호/주소 수정

---

## [2. 상품 관리 - 간단하게]

### 카테고리 (3단계만)
```
├─ 신선식품 (과일, 채소, 정육, 수산)
├─ 가공식품 (라면, 냉동식품, 음료)
└─ 생활용품 (세제, 화장지)
```

### 상품 정보
* **기본 정보**:
  - 상품명
  - 가격
  - 대표 이미지 1장
  - 간단한 설명 (1-2줄)
* **재고 표시**: 
  - 판매중 / 품절
  - 재고 수량은 관리자만 확인

### 검색
* **간단한 검색**: 상품명으로만 검색
* **필터**: 카테고리별 보기

---

## [3. 주문 시스템 - 핵심만]

### 장바구니
* 상품 담기 / 수량 조절 / 삭제
* 총 금액 표시
* **즉시 구매** 버튼

### 주문하기
**배송 정보 입력:**
```
1. 이름
2. 전화번호
3. 주소 (도로명 주소 - 우편번호 API 사용)
4. 상세 주소 (동/호수)
5. 배송 요청사항 (선택)
```

**배송 방법 선택:**
* 배달 받기: 배달비 3,000원 (30,000원 이상 무료)
* 매장 픽업: 무료 (2시간 후 픽업 가능)

**결제 수단:**
* 만나서 결제 (현금/카드)
* 온라인 결제 (토스페이먼츠)

### 주문 완료
* 주문번호 발급
* 문자(SMS) 자동 발송: "주문이 접수되었습니다"
* 주문 내역 페이지로 이동

---

## [4. 주문 관리 (사용자)]

### 주문 내역
* 주문 번호
* 주문 날짜/시간
* 주문 상품 목록
* 배송 방법
* 주문 상태:
  - 주문 접수 → 준비중 → 배달중 → 완료
  - 픽업 대기 → 픽업 완료

### 주문 취소
* **조건**: 준비중 상태일 때만 가능
* 전화로 취소 안내 (자동화 X)

---

## [5. 관리자 페이지 - 필수 기능만]

### 로그인
* 관리자 ID/PW (하드코딩 가능)

### 상품 관리
```
[상품 목록]
- 상품 검색
- 품절 처리 버튼

[상품 등록/수정]
- 상품명 입력
- 가격 입력
- 카테고리 선택
- 이미지 업로드 (1장)
- 재고 수량 입력
```

### 주문 관리
```
[오늘의 주문]
- 신규 주문 알림 (소리)
- 주문 목록 (최신순)
- 상태 변경 버튼:
  [주문접수] → [준비중] → [배달중] → [완료]

[주문 상세]
- 고객 정보 (이름, 전화, 주소)
- 주문 상품 목록
- 총 금액
- 배송 방법
- 메모란 (사장님 전용)
```

### 간단한 통계
* 오늘 주문 건수
* 오늘 매출
* 이번 주 매출

---

## [6. 알림 시스템 - 최소화]

### 고객 알림
* **주문 접수**: SMS 1회 발송 (필수)
* **픽업 준비 완료**: SMS 발송
* 이메일은 생략

### 관리자 알림
* **신규 주문**: 
  - 관리자 페이지에 팝업
  - 알림음 재생
  - 카카오톡 알림 (선택)

* **컬러**: 
  - Primary: 초록색 (#4CAF50) - 신선함
  - Secondary: 주황색 - 강조
  - Background: 흰색/연한 회색

│ │     │     │       │

│ │ 상품 │ 상품 │       │

* 화면 하단에 장바구니 아이콘 + 개수 표시
* 클릭 시 장바구니 페이지로 이동

# 제약사항

## [운영 조건]
* **배달 지역**: 마트 반경 2km 이내만
* **영업 시간**: 오전 9시 - 오후 9시
* **최소 주문금액**: 10,000원
* **배달비**: 3,000원 (30,000원 이상 무료)

## [기술 제약]
* 초기에는 **신선식품 사진은 간단히** 촬영
* 재고 관리는 수동 (자동화 X)
* 결제 실패 시 전화 안내

# 개발 가이드

## [개발 순서]
```
1주차: 상품 목록/상세 + 장바구니
2주차: 주문하기 + 관리자 페이지
3주차: 회원 기능 + SMS 연동
4주차: 온라인 결제 + 테스트
```

## [데이터베이스 구조 (간단)]

### Users (회원)
```
id, name, phone, password, address, created_at
```

### Products (상품)
```
id, name, price, category, image_url, stock, is_active, created_at
```

### Orders (주문)
```
id, user_name, user_phone, address, delivery_type, 
payment_method, total_amount, status, created_at
```

### OrderItems (주문 상품)
```
id, order_id, product_id, quantity, price
```
