# 필마트 관리자 매뉴얼 (사장님용)

## 1. 관리자 페이지 접속

### 접속 방법
1. 인터넷 브라우저에서 **필마트주소/admin.html** 로 접속
2. 아이디와 비밀번호 입력
   - 아이디: `admin`
   - 비밀번호: `pilmart2024`
3. **"로그인"** 버튼 클릭

---

## 2. 대시보드 (메인 화면)

로그인하면 대시보드가 나타납니다.

### 한눈에 보는 정보
- **오늘 주문**: 오늘 들어온 주문 건수
- **오늘 매출**: 오늘 총 매출액
- **이번 주 매출**: 이번 주 총 매출액
- **신규 주문**: 아직 처리하지 않은 새 주문

### 최근 주문
- 가장 최근 5개의 주문이 표시됩니다.
- 클릭하면 주문 상세 정보를 볼 수 있습니다.

---

## 3. 주문 관리

왼쪽 메뉴에서 **📋 주문 관리**를 클릭합니다.

### 주문 목록 보기
- 모든 주문이 최신순으로 표시됩니다.
- 상태별로 필터링 가능 (주문접수, 준비중, 배달중 등)
- 날짜별로 필터링 가능

### 주문 상세 보기
주문 카드를 클릭하면 상세 정보가 나타납니다:
- 고객 정보 (이름, 전화번호, 주소)
- 주문 상품 목록
- 결제 금액
- 배송 방법 (배달/픽업)

### 주문 상태 변경
주문 상세 화면에서 상태를 변경할 수 있습니다:

**배달 주문:**
1. **주문 접수** → 주문이 들어왔음
2. **준비중** → 상품을 준비하고 있음
3. **배달중** → 배달이 시작됨
4. **완료** → 배달 완료

**픽업 주문:**
1. **주문 접수** → 주문이 들어왔음
2. **준비중** → 상품을 준비하고 있음
3. **픽업 대기** → 고객이 찾아가기를 기다림
4. **픽업 완료** → 고객이 상품을 가져감

### 메모 기능
- 각 주문에 메모를 남길 수 있습니다.
- 예: "양파 대신 감자로 변경 요청"
- 메모는 사장님만 볼 수 있습니다.

### 고객에게 전화하기
- 고객 연락처를 클릭하면 바로 전화할 수 있습니다.

---

## 4. 상품 관리

왼쪽 메뉴에서 **📦 상품 관리**를 클릭합니다.

### 상품 목록
- 등록된 모든 상품이 표시됩니다.
- 상품명으로 검색할 수 있습니다.

### 새 상품 등록
1. **"+ 상품 등록"** 버튼 클릭
2. 상품 정보 입력:
   - 상품명 (필수)
   - 가격 (필수)
   - 카테고리 선택 (신선식품/가공식품/생활용품)
   - 설명 (선택)
   - 재고 수량
   - 상품 이미지 (선택)
3. **"저장"** 버튼 클릭

### 상품 수정
1. 상품 목록에서 **"수정"** 버튼 클릭
2. 정보를 수정한 후 **"저장"** 클릭

### 품절 처리
- 상품이 품절되면 **"품절처리"** 버튼을 클릭합니다.
- 품절된 상품은 고객에게 "품절"로 표시됩니다.
- 재입고되면 **"재입고"** 버튼을 클릭합니다.

### 상품 이미지 업로드
- JPG, PNG 파일을 업로드할 수 있습니다.
- 권장 크기: 500 x 500 픽셀
- 정사각형 이미지가 가장 보기 좋습니다.

---

## 5. 신규 주문 알림

### 알림 방식
- 새 주문이 들어오면 **알림음**이 울립니다.
- 화면에 **팝업 알림**이 나타납니다.
- 주문 관리 메뉴에 숫자 배지가 표시됩니다.

### 알림 확인
- 30초마다 자동으로 새 주문을 확인합니다.
- **"새로고침"** 버튼으로 수동 확인도 가능합니다.

---

## 6. 자주 묻는 질문

### Q. 비밀번호를 변경하고 싶어요.
A. 현재는 수동으로 변경해야 합니다. 개발자에게 문의해주세요.

### Q. 주문을 취소하려면?
A. 주문 상세에서 **"취소됨"** 상태로 변경하세요.

### Q. 상품을 삭제하려면?
A. 상품은 삭제되지 않고 "비활성" 상태로 변경됩니다.
   상품 수정 화면에서 "판매중" 체크를 해제하세요.

### Q. 매출 데이터를 더 자세히 보고 싶어요.
A. 대시보드에서 기본 통계를 확인할 수 있습니다.
   더 자세한 분석이 필요하면 개발자에게 문의해주세요.

---

## 7. 문제 해결

### 페이지가 안 열려요
- 인터넷 연결을 확인해주세요.
- 브라우저를 새로고침 해보세요 (F5 또는 Ctrl+R)

### 로그인이 안 돼요
- 아이디와 비밀번호를 정확히 입력했는지 확인해주세요.
- Caps Lock이 켜져있는지 확인해주세요.

### 주문이 안 보여요
- **"새로고침"** 버튼을 눌러보세요.
- 필터가 설정되어 있는지 확인해주세요.
